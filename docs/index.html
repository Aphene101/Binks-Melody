jamendo.js

Jamendo API client + player/visualizer/control wiring and playlist add/remove UI. This module: - searches Jamendo tracks, - renders results and dispatches "play-request" events, - manages the global audio player (play/pause/seek/next/prev), - draws a canvas-based bar visualizer, - provides add/remove to playlist choosers backed by Firestore, - exposes minimal globals for cross-page "Add to playlist" reuse.
